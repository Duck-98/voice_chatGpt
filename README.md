### 음성으로 ChatGpt를 사용하는 웹

<br/>[시연 영상](https://www.youtube.com/watch?v=RTvRZ5OCWLM)

#### 기술 스택

1. React
2. Typescript
3. react-router-dom
4. styled-components
5. react-speech-recognition (음성 인식)
6. openai

#### 사용방법

깃 클론을 한다면

```
1. git clone을 한다.
2. API key를 https://platform.openai.com/ 에 들어가서 받은 후 env 파일에 입력한다.
3. npm start
```
